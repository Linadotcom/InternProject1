Incident Management System

Overview
A full-stack web application designed to track and manage infrastructure incidents within a data center environment.

Features:
- Track and manage infrastructure incidents
- Real-time monitoring dashboard
- Structured data handling for system events
- Visualization of incident metrics using Chart.js

Tech Stack:
- Backend and Frontend: Django (django templates)
- Database: sqlite
- Visualization: Chart.js

Note
This project was developed as part of an internship at Atlas Cloud Services and focuses on demonstrating
system design and full-stack development concepts.
